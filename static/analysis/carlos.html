Análise 1: Os dados são confiáveis ?¶

A proposta dessa anélise e verificar a veracidade desses dados , isto é ,dado uma População , ao examinarmos um conjunto de amostras , a maioria delas vai seguir o mesmo padrão ? Ou seja, já sabendo os possíveis resultados futuros da População essas essas amostras vão obter os mesmos resultados ? Será que os resultados estarão "bem distríbuidos" de forma uniforme ? Dessa forma poderemos garantir a confiabilidade dos dados e que eles seguem um padrão praticamente uniforme.

def outliers(df,ft):
    q1= df[ft].quantile(.25)
    q3= df[ft].quantile(.75)
    print("Q1: {}, Q3: {}".format(q1, q3))
    iqr= q3-q1
    
    lower_bound= q1-(1.5*iqr)
    upper_bound= q3+(1.5*iqr)
    
    filtro = (df[ft] < lower_bound) | (df[ft] > upper_bound)
    ls = df[filtro]
    print(ls[ft])
    return list(ls[ft])

from numpy import mean, std, percentile
def reject_outliers(data):
    q25, q75 = percentile(data, 25), percentile(data, 75)
    iqr = q75 - q25
    cut_off = iqr * 1.5
    lower, upper = q25 - cut_off, q75 + cut_off
    return [x for x in list(data) if x > lower and x < upper]
